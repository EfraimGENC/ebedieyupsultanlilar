# Ebedi Eyüpsultanlılar AI Tools
İçerik oluşturma ve yönetim araçları.

## Kurulum
Bu araçları kullanmak için aşağıdaki komutu çalıştırarak gerekli bağımlılıkları yükleyin:

```bash
uv sync --frozen
```

## Kullanım
Araçları kullanmak için terminalde aşağıdaki komutu çalıştırabilirsiniz:
```bash
uv run python main.py
```
